# n8n AI Agent with GROQ and SERP API

Chat-triggered AI agent using n8n, GROQ (via API), and SERP API for web-based information retrieval and response generation.

**Features:**

* Chat interaction.
* Web search (SERP API).
* AI processing (GROQ).
* n8n automation.

**Setup:**

1.  Clone repo.
2.  Install n8n.
3.  Import workflow.
4.  Add API keys.
5.  Run.

**Usage:**

Chat with the agent.

**Technologies:**

* n8n
* SERP API
* GROQ (API)
